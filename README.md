##  Project Overview
ShopSpark is a fully functional e-commerce SPA where users can browse products, filter by category, sort by price or rating, add items to cart, and proceed to checkout — all in Indian Rupees (₹).

##  React Concepts Used
- **JSX** — Used throughout all components
- **Functional Components with Props** — ProductCard, CartItem, CartDrawer, Badge, Toast
- **CSS Modules** — Each component has its own scoped stylesheet
- **Conditional Rendering** — Cart badge, empty cart message, product badges, toast notification
- **List Rendering** — Products grid, filter buttons, cart items using .map()

##  Tech Stack
- ReactJS
- TypeScript
- CSS Modules
- React Router
- Vite

##  Features
- 12 products across 5 categories
- Filter by category
- Sort by price and rating
- Add to cart with quantity controls
- Free delivery above ₹999
- Toast notifications
- Fully responsive on mobile and laptop
